# OpenCL versions

When enabling an OpenCL feature, all previous OpenCL verion features are enabled automatically.
There is no default OpenCL version, so you'll have to set one when importing Blaze.

| Feature | OpenCL Version |
| ------- | -------------- |
| `cl1_1` | 1.1            |
| `cl1_2` | 1.2            |
| `cl2`   | 2.0            |
| `cl2_1` | 2.1            |
| `cl2_2` | 2.2            |
| `cl3`   | 3.0            |
